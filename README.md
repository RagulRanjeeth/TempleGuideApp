# 🛕 Pancha Bhoota Temple Guide App

## 📱 Overview
The Pancha Bhoota Temple Guide App is a simple React Native mobile application that showcases the five sacred temples representing the five natural elements (Earth, Water, Fire, Air, and Space) along with their location details.

---

## 🚀 Features
- Displays all five Pancha Bhoota temples
- Represents each temple with its corresponding natural element
- Dynamic UI styling based on element type
- Scrollable and user-friendly interface
- Clean and reusable component-based architecture

---

## 🛠 Tech Stack
- React Native
- Expo
- TypeScript
